# gh org Command

Work with GitHub organizations.

## Usage

```
gh org <command> [flags]
```

## General Commands

*   `list`: List organizations for the authenticated user.

## Inherited Flags

*   `--help`: Show help for command

## Examples

```bash
$ gh org list
```
